# Dozzle\n\nDozzle is a lightweight, open-source real-time log viewer for Docker containers. It provides a clean, intuitive web interface to stream logs from all your running containers, making debugging and monitoring simple and efficient. Ideal for home lab environments and ensuring your services run smoothly.\n\n## Prerequisites\n\n*   A Docker installation.\n*   `docker-compose` (or Docker Engine with Compose V2 support).\n*   Access to the Docker socket (typically `/var/run/docker.sock`).\n\n## Docker Compose Script\n\n```yaml\nversion: '3.8'\n\nservices:\n  dozzle:\n    image: amir20/dozzle:latest\n    container_name: dozzle\n    ports:\n      - "8080:8080"\n    volumes:\n      # Mount the Docker socket to allow Dozzle to read container logs\n      - /var/run/docker.sock:/var/run/docker.sock:ro\n    restart: unless-stopped\n    # Optional: Add labels for Traefik or other proxy if applicable\n    # labels:\n    #   - "traefik.enable=true"\n    #   - "traefik.http.routers.dozzle.rule=Host(`dozzle.yourdomain.com`)"\n    #   - "traefik.http.routers.dozzle.entrypoints=websecure"\n    #   - "traefik.http.routers.dozzle.tls.certresolver=myresolver"\n```
