# Uptime Kuma\n\nUptime Kuma is a free and open-source monitoring tool. Monitor your websites, services, and APIs with real-time notifications and a beautiful, user-friendly dashboard.\n\n## Prerequisites\n\n*   Standard Docker installation.\n*   A persistent volume for data storage (e.g., a local directory mapped to `/app/data` inside the container).\n*   An available port (e.g., `3001`) on your host machine to access the Uptime Kuma web interface.\n\n## Docker Compose Script\n\n```yaml\nversion: '3.8'\n\nservices:\n  uptime-kuma:\n    image: louislam/uptime-kuma:1\n    container_name: uptime-kuma\n    volumes:\n      - ./uptime-kuma-data:/app/data\n    ports:\n      - "3001:3001"\n    restart: unless-stopped\n    # Optional: Set a specific user/group for volume permissions (e.g., to match host user)\n    # user: "1000:1000"\n    # Optional: Network if you have a custom bridge or reverse proxy setup\n    # networks:\n    #   - default\n\n# Optional: Define custom networks if needed (uncomment and adjust)\n# networks:\n#   default:\n#     external: true\n#     name: my-custom-network\n```
